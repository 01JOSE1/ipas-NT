# IPAS - Proyecto Nuevas tecnologias

## Compilación y Build del Proyecto

El proyecto utiliza Maven como herramienta de build y gestión de dependencias.

Para compilar y construir el proyecto, ejecuta el siguiente comando en la raíz del proyecto:

```
mvn clean install
```

Esto descargará las dependencias, compilará el código y generará los artefactos necesarios.

### Requisitos
- Java 17 o superior
- Maven instalado

Si tienes dudas sobre el proceso de build, consulta la documentación oficial de Maven: https://maven.apache.org/guides/getting-started/
